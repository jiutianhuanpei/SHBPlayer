# CustomPlayerController
自定义视频播放器

*本播放器功能 ：

*可播放本地和网络视频，暂停、继续播放，拖动滑块快进或快退播放进度，点击视频空白区域显隐导航栏及工具栏，点击视频区域播放或暂停视频，调节系统音量
